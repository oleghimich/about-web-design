Коротко про вебдизайн

HTML

HTML (англ. HyperText Markup Language — мова розмітки гіпертексту) — стандартизована мова розмітки документів для перегляду вебсторінок у браузері. Браузери отримують HTML-документ від сервера за протоколами HTTP/HTTPS або відкривають з локального диска, далі інтерпретують код в інтерфейс, який відображається на екрані. Елементи HTML є будівельними блоками сторінок. Зображення, форми та інші об'єкти можуть бути вбудовані у візуалізовану сторінку. HTML позначає структурну семантику тексту, наприклад заголовки, абзаци, списки, посилання, цитати та інше. Теги HTML можуть бути парними (наприклад <p>) або самозакривальними (наприклад <img />). В HTML також можна вбудовувати скрипти, такі як JavaScript, а зовнішній вигляд задається за допомогою CSS.

Детальніше

CSS

CSS (англ. Cascading Style Sheets, укр. Каскадні таблиці стилів) — мова стилів, що використовується для опису вигляду вебсторінок. CSS є однією з основних технологій веба поряд із HTML та JavaScript. Вона дозволяє візуально форматувати елементи сторінки, змінювати кольори, шрифти, розташування та інші властивості. Існують різні рівні CSS: CSS1, CSS2, CSS3. Профілі CSS адаптовані для різних пристроїв — принтерів, мобільних пристроїв тощо. CSS замінив табличне верстання на сучасне блокове, де відокремлюється структура (HTML) від дизайну (CSS).

Детальніше

JS

JavaScript (JS) — динамічна, об'єктно-орієнтована прототипна мова програмування. Використовується для написання сценаріїв вебсторінок, що дозволяє взаємодіяти з користувачем, керувати браузером, надсилати/отримувати дані з сервера без перезавантаження сторінки. JavaScript має динамічну типізацію, підтримує різні парадигми (в тому числі функціональну) та широко використовується у веброзробці.

Детальніше

Теги

html
Визначає початок та кінець HTML-документа.
head
Визначає метадані вебсторінки, такі як заголовок, ключові слова тощо.
title
Встановлює заголовок сторінки, який показується у вкладці браузера.
body
Визначає основний вміст вебсторінки, який бачить користувач.
